NovaGest
Detalles

Categoria - Automatización de procesos

Tiempo de desarollo - 3 meses

NovaGest es una plataforma interna diseñada para digitalizar y optimizar la gestión de clientes, facturación y reportes de una empresa. Incluye flujos automáticos, alertas inteligentes y paneles visuales personalizados.

Tecnologias usadas
  • React.js
  • Tailwind CSS
  • Node.js
  • Express
  • MongoDB
  • Api Rest
  • Google Auth
Miembros del equipo

Implementar NovaGest fue un antes y un después para nuestra empresa. Pasamos de planillas dispersas y procesos manuales a tener todo centralizado, automatizado y en tiempo real. La plataforma es muy intuitiva, los paneles de control nos permiten visualizar todo al instante y el equipo de TecnoNova estuvo siempre atento a nuestras necesidades. En menos de tres meses ya teníamos todo funcionando y desde entonces nuestra productividad mejoró notablemente.

Carla Méndez Coordinadora Administrativa, Grupo Synet 13/03/2025
